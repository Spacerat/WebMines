    Some words on AjaxMines

    AjaxMines is a collaborative minesweeper game that I made over 3 years ago, as my first adventure in to the world of creating multiplayer web-based games. Although it is functional, and I never fully completed working on it once I'd completed the basic engine. Still, it's here today for historical reasons, and also because it's still kind of cool. Even today if I wanted a game of multiplayer minesweeper I might choose it over other services which require that you register an account or sit through ads!

    How to play

    1. Set a player name
    2. Join a game if one exists by clicking on its link
    3. Otherrwise, create one! Give it a name, set some settings, click Create, then send people the URL so they can join you.

    Known bugs and omissions

    • Completed games often take a while to disappear from the "Select game" menu
    • Sometimes clicks in game don't get synchronised between all players. Reloading the page fixes this
    • You have to refresh the homepage to reload the game list
    • The website is ugly as sin!
